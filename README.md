# Assets Vault

Assets Vault is my personal asset management application for browsing and searching reusable design assets.

The goal is simple: instead of searching the internet every time I need an icon, emoji, font, or logo, I keep a curated collection in one place that is easy to explore and reuse across my projects.

This application is intentionally lightweight. It is a static website built with HTML, CSS, and JavaScript, designed for local use and GitHub Pages.
